「Fresh Sends（フレッシュ・センズ）」のウェブサイトは、従来の「古臭い花屋さん」のイメージを完全に覆す、**Z世代・ミレニアル世代向けのモダンなD2C（Direct to Consumer）ライフスタイルブランド**のデザインを採用しています。

このサイトのデザインをフロントエンドエンジニアやUI/UXデザイナー、あるいはAIが完全に再現できるように、**デザインシステム・仕様書（プロンプト）形式**で言語化しました。

このまま開発指示書やデザインプロンプトとしてお使いいただけます。

# 🌸 Fresh Sends クローン：デザイン仕様＆実装ガイド

## 1. 全体的な世界観とトーン＆マナー（コンセプト）

* **スタイル:** 「ネオ・ブルータリズム」をベースにした、ポップで洗練されたD2Cスタイル。
* **キーワード:** チアフル（陽気）、エディトリアル（雑誌風）、大胆、シンプル、親しみやすさ。
* **UIの特徴:** 直線的なグリッド、はっきりとした境界線（ボーダー）、大きなタイポグラフィ、余白（ホワイトスペース）の大胆な活用。

## 2. カラーパレット（色彩設計）

真っ白な背景は避け、少し温かみのある色をベースにすることで、ビビッドなアクセントカラーを引き立たせます。

* **ベース背景色 (Background):** `#F9EAE6` (ごく薄いピーチベージュ) または `#F5F3EF` (温かみのあるオフホワイト)。画面全体が柔らかい印象になるように。
* **メインテキスト (Primary Text):** `#191717` または `#090605` (真っ黒ではない、インクのような濃いチャコール/ダークブラウン)。
* **ブランド・アクセントカラー (Accent Colors):**
* **ポピーレッド / ウォームオレンジ:** `#EA5B3B` （ボタン、セール通知、ホバー時のアクセントなどに使用）
* **パステルピンク / 鮮やかなマゼンタ:** 限定パッケージやバナーの背景色として使用。


* **UIボーダー (Borders):** `#191717` の **1px〜2pxのソリッドな実線** を多用し、ヘッダー、セクション、ボタン、カードを物理的なブロックのように区切る。

## 3. タイポグラフィ（フォント指定）

文字そのものを「デザインの主役」として扱います。

* **見出し（Headings / Display Font）:**
* **特徴:** チャンキー（太め）でモダンな「セリフ体（明朝体系）」。エディトリアルな高級感と遊び心を両立させる。
* **代替フォント例:** `Playfair Display`, `Ogg`, `GT Super`, `Recoleta`
* **スタイル:** 非常に大きく、太字。時にはイタリック（斜体）を混ぜてアクセントにする。


* **本文・UIテキスト（Body / UI Font）:**
* **特徴:** 視認性が高く、クリーンで幾何学的な「サンセリフ体（ゴシック体系）」。
* **代替フォント例:** `Inter`, `Helvetica Neue`, `Futura`
* **スタイル:**
* **ナビゲーションやボタン (CTA):** 全て大文字（UPPERCASE）で太字。文字間隔（letter-spacing）を広めに取る（例: `0.05em`）。




* **絵文字の活用:** 見出しやボタンテキストの末尾に、意図的に絵文字（☕、💐、📦、⭐）を配置して親しみやすさを出す。

## 4. UIコンポーネントとレイアウト（構造）

* **ヘッダー（ナビゲーションバー）:**
* 上部に固定（Sticky）され、下部に太い黒のボーダー（`border-bottom: 2px solid #191717`）を引く。
* 左側にブランドロゴ（または太字のテキストロゴ）、中央または右側に「SEND FLOWERS」「FLOWERS」「NOT FLOWERS」など、全て大文字の直感的なリンクを配置。


* **マーキー（ティッカーテキスト）アニメーション:**
* 画面の最上部（告知バー）やセクションの区切りに、CSSアニメーションで「テキストが無限に横スクロールする帯（Marquee）」を配置する。
* 例: `[ KICK THE WINTER BLUES WITH THE COFFEE SEND ☕ KICK THE WINTER BLUES... ]`


* **ボタン（Call to Action）:**
* 角は少し丸める程度（`border-radius: 4px` 〜 `8px`）、または完全なピル型（丸形）。
* 通常時は「背景色レッド × テキスト白」か「背景色ベージュ × 黒の太いボーダー」。
* ホバーエフェクトは少し大胆に。背景色が反転する、またはボタン全体が少し上に浮き上がり、下にあるドロップシャドウが太くなるようなインタラクション。


* **商品カード（Product Cards）:**
* 黒いボーダーで囲まれたカード型レイアウト。
* 上部に大きな正方形の商品画像、下部に左揃えで「商品名（セリフ体）」「価格」「短い説明文」「Learn More ボタン」。



## 5. アートディレクション（写真・画像）

* **アンボクシング（開封）体験の強調:** ただの花の接写ではなく、「オリジナルデザインの箱に入っている状態」や「ブランド特製の英字新聞風ラッピング材で包まれた状態」の写真を多用する。
* **ライティング:** スタジオで撮影された、影がはっきり出るような明るくパキッとした自然光風のフラッシュライティング。
* **フル幅ヒーロー画像:** トップページには画面幅いっぱい（Full-width）の高品質なカルーセル画像を配置し、その上に大胆なコピーを乗せる。

## 6. コピーライティング（テキストのトーン）

* **フランクでウィットに富んだ言葉遣い:**
* 「Not just another flower delivery service.（ただのフラワーデリバリーじゃないよ）」
* 「Always fresh!（いつでも新鮮！）」
* 「Consider it sent.（もう送ったも同然）」


* 堅苦しい「ご注文はこちら」ではなく、「SEND FLOWERS（花を贈る）」「SHOP BESTSELLERS（人気商品を見る）」など、直接的なアクションを促す言葉。

### 💻 開発者へ向けた実装のポイント（CSSヒント）

このデザインを再現する際、以下のCSSルールをベースにすると「Fresh Sendsらしさ」が一気に出ます。

```css
:root {
  --bg-color: #F9EAE6; /* 温かい背景色 */
  --text-main: #191717; /* 濃いチャコール */
  --brand-accent: #EA5B3B; /* ポピーレッド */
  --border-style: 2px solid var(--text-main); /* 特徴的なブルータリズム境界線 */
}

body {
  background-color: var(--bg-color);
  color: var(--text-main);
  font-family: 'Helvetica Neue', Inter, sans-serif;
  -webkit-font-smoothing: antialiased;
}

h1, h2, h3 {
  font-family: 'Playfair Display', serif; /* チャンキーなセリフ体を適用 */
  font-weight: 800;
  letter-spacing: -0.02em; /* 見出しは少し詰めて塊感を出す */
}

.button, .nav-link {
  text-transform: uppercase; /* 全て大文字 */
  font-weight: bold;
  letter-spacing: 0.05em; /* ボタンやナビは少し広げる */
}

/* 特徴的なカードレイアウトのスタイル */
.product-card {
  border: var(--border-style);
  border-radius: 8px;
  overflow: hidden;
  background-color: #fff;
  transition: transform 0.2s ease, box-shadow 0.2s ease;
}

.product-card:hover {
  transform: translateY(-4px);
  box-shadow: 4px 4px 0px var(--text-main); /* ホバー時のソリッドな影 */
}

```

この仕様書をClaudeやCursor、v0などのAIコーディングツールに渡し、**「このガイドラインに従ってReact/Next.jsとTailwind CSSでトップページをコーディングしてください」**と指示すれば、極めて精度の高いクローンサイトが生成されます。
